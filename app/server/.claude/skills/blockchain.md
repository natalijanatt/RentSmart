# Skill: Blockchain Module (Solana)

## Context

Optional verification and escrow layer using Solana Devnet with Anchor framework. Stores contract hashes on-chain, manages deposit escrow in a PDA (Program Derived Address), and executes settlement distribution. This module is a LEAF — it never calls other modules. Other modules call INTO blockchain at 5 key moments.

CRITICAL: Solana is NOT critical path. Every call is wrapped in try/catch. If Solana fails, the operation continues — log a warning and move on.

## When Solana is called (5 moments only)

| Moment | Caller | Solana instruction | What's stored |
|--------|--------|--------------------|---------------|
| Contract created | contracts.service.create() | initialize() | Contract hash, deposit amount |
| Tenant accepts | contracts.service.accept() | lock_deposit() | Tenant sends SOL to PDA |
| Check-in approved | inspections.service.approve('checkin') | record_checkin() | Combined image hash |
| Check-out approved | inspections.service.approve('checkout') | record_checkout() | Combined image hash |
| Settlement finalized | settlement approval (both sides) | execute_settlement() | Settlement hash, releases escrow |

Everything between (rejections, re-uploads, LLM analysis, rule engine) is OFF-CHAIN.

## Service: solana.service.ts

```typescript
import { Connection, PublicKey, Keypair, SystemProgram } from '@solana/web3.js';
import { AnchorProvider, Program, Wallet, BN } from '@coral-xyz/anchor';
import { env } from '../../config/env';
import { sha256 } from '../../shared/utils/hash';

// EUR to lamports conversion (approximate, for demo)
const EUR_TO_SOL_RATE = 0.005; // 1 EUR ≈ 0.005 SOL (for demo)
const LAMPORTS_PER_SOL = 1_000_000_000;

export class SolanaService {
  private connection: Connection;
  private authority: Keypair;
  private program: Program;
  private programId: PublicKey;

  constructor() {
    this.connection = new Connection(env.SOLANA_RPC_URL, 'confirmed');

    if (env.SOLANA_AUTHORITY_KEYPAIR) {
      const keypairBytes = JSON.parse(env.SOLANA_AUTHORITY_KEYPAIR);
      this.authority = Keypair.fromSecretKey(new Uint8Array(keypairBytes));
    }

    if (env.SOLANA_PROGRAM_ID) {
      this.programId = new PublicKey(env.SOLANA_PROGRAM_ID);
    }

    // Initialize Anchor program (requires IDL — generated by anchor build)
    // this.program = new Program(IDL, this.programId, provider);
  }

  findPDA(contractId: string): { pda: PublicKey; bump: number } {
    const [pda, bump] = PublicKey.findProgramAddressSync(
      [Buffer.from('rental'), Buffer.from(contractId)],
      this.programId
    );
    return { pda, bump };
  }

  async initializeContract(
    contractId: string,
    contractHash: Buffer,
    depositLamports: number,
    landlordPubkey: string
  ): Promise<TxResult & { pda_address: string }> {
    const { pda } = this.findPDA(contractId);

    const tx = await this.program.methods
      .initialize(
        Buffer.from(contractId.replace(/-/g, '')),
        contractHash,
        new BN(depositLamports)
      )
      .accounts({
        agreement: pda,
        landlord: new PublicKey(landlordPubkey),
        authority: this.authority.publicKey,
        systemProgram: SystemProgram.programId,
      })
      .signers([this.authority])
      .rpc();

    return {
      tx_signature: tx,
      pda_address: pda.toBase58(),
      explorer_url: `https://explorer.solana.com/tx/${tx}?cluster=devnet`,
    };
  }

  async recordCheckin(
    contractId: string,
    imageHashes: string[],
    landlordPubkey: string
  ): Promise<TxResult> {
    const { pda } = this.findPDA(contractId);
    const combinedHash = this.hashImages(imageHashes);

    const tx = await this.program.methods
      .recordCheckin(combinedHash)
      .accounts({
        agreement: pda,
        landlord: new PublicKey(landlordPubkey),
      })
      .rpc();

    return {
      tx_signature: tx,
      explorer_url: `https://explorer.solana.com/tx/${tx}?cluster=devnet`,
    };
  }

  async recordCheckout(
    contractId: string,
    imageHashes: string[],
    tenantPubkey: string
  ): Promise<TxResult> {
    const { pda } = this.findPDA(contractId);
    const combinedHash = this.hashImages(imageHashes);

    const tx = await this.program.methods
      .recordCheckout(combinedHash)
      .accounts({
        agreement: pda,
        tenant: new PublicKey(tenantPubkey),
      })
      .rpc();

    return {
      tx_signature: tx,
      explorer_url: `https://explorer.solana.com/tx/${tx}?cluster=devnet`,
    };
  }

  async executeSettlement(
    contractId: string,
    settlementHash: Buffer,
    tenantAmount: number,
    landlordAmount: number,
    tenantPubkey: string,
    landlordPubkey: string
  ): Promise<TxResult> {
    const { pda } = this.findPDA(contractId);

    const tx = await this.program.methods
      .executeSettlement(
        settlementHash,
        new BN(tenantAmount),
        new BN(landlordAmount)
      )
      .accounts({
        agreement: pda,
        tenant: new PublicKey(tenantPubkey),
        landlord: new PublicKey(landlordPubkey),
        authority: this.authority.publicKey,
      })
      .signers([this.authority])
      .rpc();

    return {
      tx_signature: tx,
      explorer_url: `https://explorer.solana.com/tx/${tx}?cluster=devnet`,
    };
  }

  async getAgreement(contractId: string): Promise<SolanaAgreement | null> {
    try {
      const { pda } = this.findPDA(contractId);
      const account = await this.program.account.rentalAgreement.fetch(pda);
      return account as unknown as SolanaAgreement;
    } catch {
      return null; // PDA doesn't exist yet
    }
  }

  hashImages(imageHashes: string[]): Buffer {
    const combined = imageHashes.sort().join('');
    return Buffer.from(sha256(combined), 'hex');
  }

  eurToLamports(eurAmount: number): number {
    return Math.round(eurAmount * EUR_TO_SOL_RATE * LAMPORTS_PER_SOL);
  }
}
```

## Routes: blockchain.routes.ts

```typescript
const router = Router();

// Optional verification endpoint
router.get('/:id/blockchain',
  authMiddleware,
  asyncHandler(async (req, res) => {
    const solana = new SolanaService();
    const agreement = await solana.getAgreement(req.params.id);

    if (!agreement) {
      return res.json({ on_chain: false, message: 'No on-chain record found' });
    }

    res.json({
      on_chain: true,
      agreement,
      explorer_url: `https://explorer.solana.com/address/${solana.findPDA(req.params.id).pda}?cluster=devnet`,
    });
  })
);

export default router;
```

## Anchor program structure (Rust side — for reference)

The Solana program (built with Anchor) should have these instructions:
- `initialize(contract_id: [u8; 32], contract_hash: [u8; 32], deposit_lamports: u64)` — creates PDA
- `lock_deposit()` — tenant transfers SOL to PDA
- `record_checkin(image_hash: [u8; 32])` — stores check-in image hash
- `record_checkout(image_hash: [u8; 32])` — stores check-out image hash
- `execute_settlement(settlement_hash: [u8; 32], tenant_amount: u64, landlord_amount: u64)` — distributes escrow

PDA account structure:
```rust
#[account]
pub struct RentalAgreement {
    pub contract_id: [u8; 32],       // UUID bez crtica (32 ASCII karaktera)
    pub contract_hash: [u8; 32],     // SHA-256 ugovora
    pub deposit_lamports: u64,
    pub landlord: Pubkey,
    pub tenant: Pubkey,
    pub state: AgreementState,       // Created=0, DepositLocked=1, CheckinRecorded=2, CheckoutRecorded=3, Settled=4
    pub checkin_hash: [u8; 32],
    pub checkout_hash: [u8; 32],
    pub settlement_hash: [u8; 32],
    pub bump: u8,
}
```

## DO

- DO wrap EVERY Solana call in try/catch — failures must not break the flow
- DO log audit events for successful Solana transactions (with tx hash and explorer URL)
- DO store tx_signature and PDA address in the contracts table
- DO use `findProgramAddressSync` for PDA derivation — deterministic
- DO sort image hashes before combining — ensures deterministic combined hash
- DO convert EUR to lamports using a fixed rate (demo) or oracle (production)

## NEVER

- NEVER let a Solana failure crash the request — catch, warn, continue
- NEVER put personal data on-chain (names, phone numbers, addresses)
- NEVER call Solana for intermediate events (rejections, re-uploads, LLM analysis)
- NEVER import from other modules — blockchain is a leaf
- NEVER store the authority private key in code — use env var
- NEVER use mainnet for MVP — Devnet only
